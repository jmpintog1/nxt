
# Robótica Educativa con LEGO NXT: Del Código a la Física Aplicada 🚀

Bienvenido a un enfoque completamente nuevo para la enseñanza de las ciencias y la tecnología. 

Este libro digital no es un simple manual de instrucciones para armar bloques. Es un ecosistema de aprendizaje diseñado para transformar los kits **LEGO Mindstorms NXT** en verdaderos instrumentos de laboratorio. A través del lenguaje de programación **NXC (Not eXactly C)**, dejamos atrás los bloques visuales infantiles para introducir a los estudiantes en la programación estructurada real, resolviendo problemas físicos del mundo concreto.

---

## 🎯 ¿Para quién es este libro?

### 👨‍🏫 Para Profesores y Colegios (Solución "Llave en Mano")
Sabemos que el mayor obstáculo de la robótica escolar es la fricción técnica y la falta de tiempo para planificar. Este libro está estructurado para ser tu bitácora de clases directa al aula:
* **Supervivencia Técnica:** Incluye guías detalladas para evadir las restricciones de Windows modernos e instalar los drivers y el entorno *Bricx Command Center* sin frustraciones.
* **Alineación Curricular:** Cada desafío está vinculado a los Objetivos de Aprendizaje (OA) del MINEDUC en Ciencias, Física y Matemáticas.
* **Simulaciones Interactivas:** Cada capítulo incluye simuladores web para que los alumnos entiendan el concepto matemático antes de tocar el hardware.

### 👩‍💻 Para Estudiantes (Ingeniería del Siglo XXI)
Prepárate para programar algoritmos utilizados en la industria automotriz y aeroespacial. Aquí aprenderás a:
* Escribir código en un lenguaje basado en C.
* Utilizar sensores para leer el entorno en tiempo real.
* Extraer telemetría (Datalogging) directo desde el microprocesador del robot hacia Excel para analizar tus propios gráficos de movimiento.

## 🛠️ Explora nuestros Laboratorios

Nuestras experiencias de aula están divididas por niveles y diseñadas para ser completamente rupturistas:

1. **[Ecolocalización](actividad_1.html)** *(Enseñanza Básica)*
   Biomímesis en acción. Utiliza ondas de sonido y el sensor ultrasónico para crear un sistema de ecolocalización proporcional.
2. **[Control de Inercia](actividad_2.html)** *(Enseñanza Básica)*
   La Primera Ley de Newton puesta a prueba. Domina los bucles de programación para vencer la inercia y crear curvas de aceleración perfectas.
3. **[Telemetría MRU](actividad_3.html)** *(Enseñanza Media)*
   ¿Existe la velocidad constante? Extrae los datos de los tacómetros internos del robot hacia un archivo CSV y descubre la verdad del roce usando gráficas matemáticas.
4. **[Frenado Autónomo](actividad_4.html)** *(Enseñanza Media)*
   Introducción a la Teoría de Control. Programa un "Controlador Proporcional" idéntico al control de crucero adaptativo de los vehículos modernos.
